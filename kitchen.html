Restaurant Site

Overview

This webpage shows a list of menu items of a restaurant and consists of an admin page where the admin can login and edit the items shown on the page in case of promotions where they may be adding new dishes or removing some dishs to introduce a new one. Users can make reviews on the dishs they have tried from a range of 1 - 5. 5 being excellent and 1 being bad.

Admin Login Page

This is the login page where the admin can login into the admin side of the website and make changes that will be reflected on the main page. If the user is not able to login to the system, they will not be able to access any of the admin privilages and if they selected any of the admin links they will be directed to the login page

Successful login

The admin user is directed to this page when the correct login details are entered and now they can edit other components of the website from the navigation bar

Wrong Username

This shows the error message that appears when the user enters a wrong username

Wrong Password

This shows the error message that appears when the user enters the wrong password for an existing username

Create Admin Account

This is a page that allows an admin to create admin accounts and it can only be accessed when an admin user is logged in

List of Admin Accounts

This page shows a list of all the current admin accounts in the datatbase. When new admin accounts are created they will be visible on this page. The admin accounts can also be deleted by pressing the delete button which will also remove the selcted account from the database

List Of Menu Items Page

This page shows a list of the current menu dishes that the restaurant offers. The admin can add a new dish, edit current dishes or delete them.

Add New Dish

On this page, the admin can add new dishes that will be shown on the main site and it will also be seen on the list of menu items page also. The reason the title of this page is "Edit Dish" is because the edit dish and add dish functionality both make use of this page but what they do is different. The add dish function adds the dish when the dish is not in the database, if the dish is already in the database, then it will be edited. This was done to reduce repetition for the code and to prevent the users from having the same dish and showing that on the site. This was done for the other edit pages too.

Add New Dish Page showing Visibility Field Options

This image is showing the possible options the admin can select regarding the visibility of a dish. This means that if the admin selected hide, the dish would not be shown on the main site for customers to see but, the admin users would be able to see it on the menu list page.

Edit Existing Dish

The admin is directed to this page when they select the edit link for a particular dish. The details of the dish are filled in into the field boxes and the admin can make changes to the dish which will be shown on the main site if the visibily is still set to show

List of categories

This page shows a list of the current categories for dishes. New categories can be added and the old ones can be edited and deleted.

Add Category Page

This page allows the admin to add new dish categories to the main site which could be used to classify new dishes.

Edit existing category

The admin is directed to this page by selected the edit link which fills the data of the selected category in the field and when the admin makes changes to it, it will be reflected on the main page

List of offers available

This is a page that allows the admin to add offers which are promotions for a new dish coming out or discounts. The admin can remove them when the offers have expired by deleting them which means that it will not be visble on the main site.

Add New Offer Page

The admin is directed to this page by selecting the add new offers link. They will need to type in a title and a description for the offer. When the offer is added the current date the offer was added on will be recorded and shown on the main site.

List of reviews and users that made them

This page is used to monitor the reviews made by the customers. Reviews can be deleted when the cutomers makes an aggressive comment that the admins would not want to visble on the site.

Homepage Navigation Bar

This image shows the navigation bar of the main site. This is the page the customers will be able to see.

Homepage Content

The homepage shows the offers that the restaurant is promoting and shows they date and the descriptions of the offers that were put up by the admin.

Selected Menu Category From Navigation Bar

When a category is selected the page shows all the dishes of the selcted category and their details. This page also shows the reviews made by other customers on the dishes of the selected category.

Review Form For Selected Categories Dishes

The users will also be able to make reviews on the dishes of the selected categories

About Us Page

This shows the about us page which describes the restaurant and also provides links to view the dishes the restaurant serves.
