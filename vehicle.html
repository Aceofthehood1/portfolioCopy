MVC Vehicle Renting System

Overview

This app allows companies to register and create an account which will allow them to view the available vehicles that they will be able to rent.They will be able to see the vehicles they have rented , their account details and when they rent a vehicle that vehicle will not be shown in the list of available vehicles to rent. Staff members will also be given a user name and password where they will be able to add ,remove and view the different types of vehicles. They will also be able to toggle the rented vehicles when companies return them so the vehicles will return to the available vehicles list

Create an account

This is the register page where the corporation user can make an account and login with the account after it's creation

Login

The user after creating an account can login into the system by typing their username and password. Staff members can also login here.

This shows the staff username and password and is the only username and password the staff members can use to login into the system
